## 二、需求說明書（SRS）

### 2.1 系統目的
本系統旨在建置一套完整且具延展性之企業人事管理系統，協助企業在員工資料、人事流程與行政管理上進行系統化管理。透過資訊系統取代傳統紙本或人工處理方式，可有效降低人為錯誤、提升資料一致性，並縮短人事作業流程時間，使企業能更專注於核心業務發展。

### 2.2 使用者角色
- **人資管理員**：負責員工基本資料維護、人事異動與薪資管理  
- **一般員工**：查詢個人資料、出勤紀錄與薪資資訊  
- **系統管理員**：負責帳號管理與權限設定  

### 2.3 功能需求
- 員工基本資料管理（新增、修改、刪除、查詢）
- 人事異動管理（到職、離職、部門異動）
- 出勤紀錄管理
- 薪資資料查詢
- 使用者登入與權限控管

### 2.4 非功能需求
- 系統回應時間低於 3 秒  
- 具備帳號驗證與權限控管機制  
- 介面操作直覺，易於學習  
- 系統模組化設計，便於維護與擴充

## 四、詳細設計說明書（LLD）

### 4.1 模組功能說明
- 使用者管理模組：登入驗證與權限控管  
- 員工資料模組：管理員工基本資料  
- 出勤管理模組：記錄上下班時間  
- 薪資管理模組：管理與查詢薪資資料  

### 4.2 核心流程（新增員工資料）
1. 人資管理員輸入員工資料  
2. 系統檢查資料完整性  
3. 資料寫入資料庫  
4. 回傳處理結果
